# DESCRIPTION
- Vue-cli demo project
- use youtube-dl to parse online video play url

## Demo
- https://urlp.heroku.com/

## Build Setup

``` bash
# install dependencies
npm install

# Development mode
# start for dev
npm run dev

# Production mode
# build [optional]
npm run build
## start server
npm start
```
